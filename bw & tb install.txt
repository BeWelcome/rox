Setup DB from https://www.bewelcome.org/svn/develstuff/trunk/testdb/ and some account for it (or if you user WAMP, you can just use root and empty password.

1. copy htdocs/.htaccess.example  -> htdocs/.htaccess
2. edit htdocs/.htaccess and modify RewriteBase (e.g./bw/htdocs/)
3. copy inc/config.inc.php.example -> inc/config.inc.php
4. edit inc/config.inc.php. Modify at least following variables:
- $db['dsn']
- $db['user']
- $db['password']
- $env['baseuri']
5. copy /bw/htdocs/bw/lib/config.inc.dist -> /bw/htdocs/bw/lib/config.inc (no need to edit the file in normal case)
6. test that everything works
