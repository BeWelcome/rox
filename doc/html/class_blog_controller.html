Processing creation of a blog.

This is a POST callback function.

Sets following errors in POST vars: title - invalid(empty) title. text - invalid(empty) text. startdate - wrongly formatted start date. enddate - wrongly formatted end date. duration - empty enddate and invalid duration. category - category is not belonging to user. trip - trip is not belonging to user. inserror - error performing db insertion. tagerror - error while updating tags.
