Detailed Description

This class stores all the information from looking up a word keycode in the database. Objects of this type are created in MOD_words::_lookup($code[, $lang]), so all the db stuff happens there. Objects of this type do not store the additional arguments from a call to $words->getFormatted($code, ..).

The main purpose is to package information for function arguments and return values, and reduce the number of single variables to deal with in a function.
